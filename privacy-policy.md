1. INTRODUCTION
═══════════════════════════════════════
 
Jour Zéro ("l'Application") est une application mobile d'aide à la sobriété. 
Cette politique de confidentialité explique comment nous collectons, utilisons 
et protégeons vos données personnelles.
 
En utilisant Jour Zéro, vous acceptez les pratiques décrites dans cette politique.
 
═══════════════════════════════════════
2. DONNÉES COLLECTÉES
═══════════════════════════════════════
 
2.1 Données stockées localement sur votre appareil
Les données suivantes sont stockées UNIQUEMENT sur votre téléphone et 
ne sont jamais transmises à nos serveurs :
 
• Votre prénom (optionnel)
• Votre date de début de sobriété
• Vos jours sobres marqués dans le calendrier
• Vos entrées de journal et humeurs
• Vos paramètres personnalisés (dépense quotidienne, etc.)
• Vos jalons personnalisés
• Votre statut d'abonnement Premium
 
2.2 Données transmises à des tiers
Lorsque vous utilisez le Coach IA, les messages que vous envoyez sont 
transmis à Anthropic (claude.ai) pour générer une réponse. 
Ces messages ne sont pas stockés par Jour Zéro.
 
Consultez la politique de confidentialité d'Anthropic :
https://www.anthropic.com/privacy
 
2.3 Données de paiement
Les paiements Premium sont traités par Google Play. 
Jour Zéro n'a jamais accès à vos coordonnées bancaires.
 
═══════════════════════════════════════
3. UTILISATION DES DONNÉES
═══════════════════════════════════════
 
Vos données sont utilisées exclusivement pour :
• Afficher votre progression de sobriété
• Calculer vos statistiques personnalisées
• Personnaliser votre expérience dans l'application
• Fournir les réponses du Coach IA
 
Nous ne vendons, ne partageons et ne transmettons jamais vos données 
personnelles à des tiers à des fins commerciales.
 
═══════════════════════════════════════
4. SÉCURITÉ DES DONNÉES
═══════════════════════════════════════
 
Toutes vos données personnelles sont stockées localement sur votre appareil 
via le système de stockage sécurisé d'Android (AsyncStorage).
 
Nous vous recommandons de :
• Protéger votre téléphone par un code PIN ou une empreinte digitale
• Effectuer des sauvegardes régulières via la fonction Export de l'application
 
═══════════════════════════════════════
5. DONNÉES DES MINEURS
═══════════════════════════════════════
 
Jour Zéro n'est pas destiné aux personnes de moins de 18 ans.
Nous ne collectons pas sciemment de données concernant des mineurs.
 
═══════════════════════════════════════
6. VOS DROITS (RGPD)
═══════════════════════════════════════
 
Conformément au Règlement Général sur la Protection des Données (RGPD), 
vous disposez des droits suivants :
 
• Droit d'accès : consultez vos données via la fonction Export
• Droit de suppression : désinstallez l'application pour supprimer 
  toutes vos données locales
• Droit de portabilité : exportez vos données au format JSON via 
  le menu Import/Export de l'application

═══════════════════════════════════════
 
Jour Zéro n'utilise aucun cookie, aucun tracker publicitaire 
et aucun outil d'analyse comportementale.
